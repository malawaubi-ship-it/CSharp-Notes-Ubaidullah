# Assignments

This folder contains the two applied final assignments completed during the C# and Data Structures
and Algorithms courses. Both assignments required designing and implementing solutions to realistic
software engineering scenarios.

---

## Assignment 1 — Data Structures and Algorithms Final Assignment

**File:** [01_ITDCA_Final_Assignment.md](01_ITDCA_Final_Assignment.md)

This assignment tested the practical ability to select an appropriate data structure or algorithm
for a given real-world scenario and implement it in C# from first principles.

| Question | Scenario | Data Structure / Algorithm |
|----------|----------|---------------------------|
| Q1 | Library book management | AVL Self-Balancing Binary Search Tree |
| Q2 | Expression / undo-redo evaluation | Stack |
| Q3 | Hospital emergency triage | Min-Heap Priority Queue |
| Q4 | Hotel search platform with configurable sorting | QuickSort with JSON deserialisation |

**Skills demonstrated:**
- Implementing self-balancing tree rotations (AVL left/right rotate)
- Heap-based priority queue with HeapifyUp and HeapifyDown
- Recursive QuickSort with configurable comparator
- JSON file deserialisation using `System.Text.Json`
- Object-oriented design with clear class separation
