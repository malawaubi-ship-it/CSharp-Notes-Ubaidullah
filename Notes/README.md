# Study Notes — C# Data Structures and Algorithms

This folder contains 20 chronological learning modules covering the full Data Structures and Algorithms
curriculum implemented in C#. Modules are ordered from the most fundamental concepts to the most advanced.

Each module includes:
- Clear concept explanations
- C# code examples demonstrating the concept
- Applied case studies where relevant
- References to authoritative sources
